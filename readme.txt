The dataset we are using is:
http://physionet.org/pn4/eegmmidb/

File and their use:
Experimental run 1: Baseline eyes open
Experimental runs 3, 7 and 11: opening left and right fist.
Within an experiment, you have 3 annotations:
T0: rest
T1: left fist
T2: right fist
